
# DEFAULT

```
/my-php-app
├── bin/                 # CLI tools and scripts
├── config/              # Configuration files
├── public/              # Web root (index.php, assets)
├── src/                 # Application source code
├── templates/           # Templates or views
├── tests/               # Automated tests
├── translations/        # Localization files
├── var/                 # Cache, logs, etc. (ignored in VCS)
├── vendor/              # Composer dependencies
├── .env                 # Environment variables
├── composer.json        # Composer package config
└── README.md            # Project documentation
```
Reference: https://roman-huliak.medium.com/how-to-structure-a-php-project-best-practices-and-real-examples-a934b44ac90d
